# Analyzing NYC Forest Restoration Sites: Size Distribution and Temporal Patterns

This repository contains a comprehensive analysis of NYC forest restoration sites, focusing on their size distribution and temporal patterns over time.

## Project Overview

This project analyzes data related to forest restoration efforts in New York City, examining:
- **Size Distribution**: Understanding the distribution of restoration site sizes
- **Temporal Patterns**: Tracking changes and trends in forest restoration activities over time
- **Geographic Insights**: Analyzing restoration site characteristics and their evolution

## Repository Contents

This repository is built with:
- **Jupyter Notebooks**: All analysis, visualizations, and documentation
- **Data Analysis**: Statistical analysis and visualization of NYC forest restoration data
- **Time Series Analysis**: Temporal pattern identification and trend analysis

## Getting Started

### Prerequisites
- Python 3.7+
- Jupyter Notebook
- Common data science libraries (pandas, numpy, matplotlib, seaborn, etc.)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Krishma-adhikari/Analyzing-NYC-Forest-Restoration-Sites-Size-Distribution-and-Temporal-Patterns.git
cd Analyzing-NYC-Forest-Restoration-Sites-Size-Distribution-and-Temporal-Patterns
```

2. Install required dependencies:
```bash
pip install -r requirements.txt
```

3. Launch Jupyter Notebook:
```bash
jupyter notebook
```
